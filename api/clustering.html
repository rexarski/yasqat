Clustering

Partitional and hierarchical clustering for sequences, plus quality indices and representative extraction. All clustering functions accept a DistanceMatrix (or its .values numpy array).

from yasqat.clustering import (
    pam_clustering, clara_clustering, hierarchical_clustering,
    pam_range, cluster_quality, extract_representatives,
    PAMClustering, HierarchicalClustering,
)

PAM — Partitioning Around Medoids

PAM (Kaufman & Rousseeuw 1990) is the standard algorithm for clustering distance matrices. It finds k medoid sequences that minimise total within-cluster dissimilarity.

pam_clustering

def pam_clustering(
    distance_matrix: np.ndarray,
    n_clusters: int,
    sequence_ids: list[int | str] | None = None,
    max_iter: int = 100,
    random_state: int | None = None,
) -> PAMClusteringResult

Parameters

Name Type Description
distance_matrix np.ndarray Square pairwise distance matrix (pass dist.values).
n_clusters int Number of clusters k.
sequence_ids list \| None Row/column labels. Defaults to integer indices.
max_iter int Maximum swap iterations.
random_state int \| None Seed for reproducible medoid initialisation.

Returns — PAMClusteringResult with attributes:

Attribute Type Description
labels np.ndarray Cluster assignment per sequence (0-indexed).
medoid_indices np.ndarray Index positions of medoid sequences.
cluster_sizes dict[int, int] Size of each cluster.

Methods on result

Method Returns Description
get_cluster_members(k) list[int \| str] Sequence IDs in cluster k.
get_medoid_ids() list[int \| str] Sequence IDs of medoids.
to_dataframe() pl.DataFrame Long-format table with sequence_id, cluster.

Example

from yasqat.clustering import pam_clustering

result = pam_clustering(
    distance_matrix=dist.values,
    n_clusters=4,
    sequence_ids=dist.labels,
    random_state=42,
)

print(result.cluster_sizes)
print(result.get_medoid_ids())

PAMClustering (class API)

class PAMClustering(n_clusters, max_iter=100, random_state=None)
    def fit(distance_matrix: np.ndarray, sequence_ids=None) -> PAMClustering
    @property labels -> np.ndarray | None
    @property medoid_indices -> np.ndarray | None
    @property result -> PAMClusteringResult | None

Scikit-learn-style class interface; equivalent to pam_clustering but reusable (call .fit() to train).

CLARA — Clustering Large Applications

CLARA (Kaufman & Rousseeuw 1990) scales PAM to large datasets by sampling. Repeatedly draws random subsets, runs PAM on each, keeps the best result.

clara_clustering

def clara_clustering(
    distance_matrix: np.ndarray,
    n_clusters: int,
    sequence_ids: list[int | str] | None = None,
    n_samples: int = 5,
    sample_size: int | None = None,
    max_iter: int = 100,
    random_state: int | None = None,
) -> PAMClusteringResult

Parameters

Name Type Default Description
n_samples int 5 Number of random sub-samples to try.
sample_size int \| None None Size of each sub-sample. Defaults to 40 + 2k.

Returns the same PAMClusteringResult as pam_clustering.

from yasqat.clustering import clara_clustering

result = clara_clustering(
    distance_matrix=dist.values,
    n_clusters=5,
    sequence_ids=dist.labels,
    n_samples=10,
    random_state=42,
)

Quality metrics

pam_range

def pam_range(
    distance_matrix: DistanceMatrix,
    k_min: int = 2,
    k_max: int = 10,
    random_state: int | None = None,
) -> pl.DataFrame

Run PAM for each k in [k_min, k_max] and compute quality metrics. Returns a DataFrame with columns k, asw, hubert_c, total_cost.

Tip

Select k by maximising asw (Average Silhouette Width). An asw > 0.5 indicates a reasonable structure; > 0.7 is strong.

quality_df = pam_range(dist, k_min=2, k_max=8)
best_k = quality_df.filter(
    pl.col("asw") == pl.col("asw").max()
)["k"][0]

cluster_quality

def cluster_quality(
    distance_matrix: np.ndarray,
    labels: np.ndarray,
) -> pl.DataFrame

Compute quality metrics for a given clustering. Returns a one-row DataFrame:

Column Description
asw Average Silhouette Width (−1 to 1; higher is better).
hubert_c Hubert’s C coefficient (0 to 1; lower is better).
total_cost Sum of within-cluster distances to medoids.

Representatives

extract_representatives

def extract_representatives(
    pool: SequencePool,
    distance_matrix: DistanceMatrix,
    labels: np.ndarray,
) -> dict[int, StateSequence]

Extract the medoid sequence for each cluster — the sequence that minimises total within-cluster dissimilarity.

Returns {cluster_id: StateSequence}.

reps = extract_representatives(pool, dist, labels=result.labels)

for cluster_id, seq in reps.items():
    print(f"Cluster {cluster_id} medoid: {seq.sequence_id}")
    print(seq.states[:10])
