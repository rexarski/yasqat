# Sunburst & tree plot sample code

Reference snippets for the two hierarchical visualizations exposed by
`yasqat.visualization`. Both render with plotnine and return a `ggplot` you
can `.save(...)` or display in a notebook.

## 1. Sunburst plot (`visualization.sunburst_plot`)

Concentric rings, one ring per position in the sequence. Arc size within a
ring is proportional to the number of sequences whose prefix-path reaches
that state at that position. Use `max_depth` to cap ring count.

```python
from yasqat.core import SequencePool
from yasqat.synthetic import generate_financial_journeys
from yasqat.visualization import sunburst_plot

df = generate_financial_journeys(n_users=500, seed=42)
pool = SequencePool(df)

p = sunburst_plot(
    pool,
    max_depth=4,
    title="Financial journey sunburst",
)
p.save("sunburst.png", dpi=150)
```

**Signature:**

```python
sunburst_plot(
    sequence: StateSequence | SequencePool,
    max_depth: int = 5,
    title: str | None = None,
    show_legend: bool | None = None,      # None = auto-hide when >15 states
    figsize: tuple[float, float] = (8, 8),
) -> ggplot
```

## 2. Sequence tree plot (`visualization.tree_plot`)

Horizontal branching tree showing the most common state-transition paths.
`min_support` prunes branches covering fewer than that fraction of
sequences; `max_depth` limits the time horizon displayed.

```python
from yasqat.core import SequencePool
from yasqat.synthetic import generate_financial_journeys
from yasqat.visualization import tree_plot

df = generate_financial_journeys(n_users=500, seed=42)
pool = SequencePool(df)

p = tree_plot(
    pool,
    max_depth=5,
    min_support=0.05,          # drop branches with <5% of sequences
    title="Financial journey branching tree",
)
p.save("sequence_tree.png", dpi=150)
```

**Signature:**

```python
tree_plot(
    sequence: StateSequence | SequencePool,
    max_depth: int = 5,
    min_support: float = 0.01,
    title: str | None = None,
    x_label: str = "Time",
    show_legend: bool | None = None,
    figsize: tuple[float, float] = (12, 8),
) -> ggplot
```

## 3. Dissimilarity tree (statistics, not a visualization)

Do **not** confuse `tree_plot` above with `statistics.dissimilarity_tree`.
The latter is a CART-style recursive partitioning of a distance matrix by
covariates — it returns a `DissTreeResult` dataclass, not a ggplot. It is
what you want for "which covariates explain trajectory differences?".

```python
import numpy as np
from yasqat.core import SequencePool
from yasqat.statistics import dissimilarity_tree
from yasqat.synthetic import generate_financial_journeys

df = generate_financial_journeys(n_users=200, seed=42)
pool = SequencePool(df)

# Use "om" (optimal matching) — handles variable-length sequences.
# Use "hamming" only if all your sequences share a fixed length.
dm = pool.compute_distances(method="om")

rng = np.random.default_rng(42)
covariates = rng.random((len(pool.sequence_ids), 2))

result = dissimilarity_tree(
    dm.values,
    covariates,
    covariate_names=["age", "income"],
    max_depth=3,
    min_node_size=10,
)

print(result)                            # DissTreeResult(n_leaves=..., root_n=...)
print("pseudo-R^2 at root:", result.root.pseudo_r2)
print("leaf assignments:", result.labels[:10])
```

`DissTreeResult` exposes the fitted tree and pseudo-R² but does not yet
have a dedicated plotnine renderer — if you need one, open a v0.4.0
enhancement issue rather than calling `tree_plot` (which only accepts a
pool/sequence, not a tree).
