Sequence analysis
for social science.

A modern Python library for categorical sequence analysis. Built on polars and plotnine — no pandas, no matplotlib API. Designed for life-course and social-science researchers.

Get Started → API Reference

from yasqat import Alphabet, SequencePool
from yasqat.metrics.optimal_matching import OptimalMatchingMetric
from yasqat.clustering import pam_clustering
from yasqat.visualization import index_plot

# Define labour-market states
alpha = Alphabet(
    states=["Employed", "Unemployed", "Education", "Inactive"],
    colors=["#0f766e", "#dc2626", "#d97706", "#71717a"],
)

# Build a pool from your data
pool = SequencePool.from_dataframe(df, alphabet=alpha)

# Compute pairwise distances
dm = OptimalMatchingMetric().pairwise(pool)

# Cluster and visualise
labels = pam_clustering(dm, k=4).labels
index_plot(pool, group=labels).draw()

Polars-native

Fast by design

No pandas dependency. All data operations run through polars — columnar, lazy, and fast even on large panel datasets with thousands of sequences.

# Returns a polars DataFrame
pool.state_distribution()

Distance metrics

Seven metrics included

Optimal Matching, Hamming, LCS, DTW, and more. Every metric outputs a DistanceMatrix — a typed numpy wrapper compatible with all clustering functions.

dm = HammingMetric().pairwise(pool)
dm.values   # numpy ndarray, shape (n, n)

TraMineR-compatible

Algorithms you already know

Algorithm correctness cross-referenced against TraMineR (R) and TanaT (Python). Naming conventions follow TraMineR so the transition from R is immediate.

turbulence(pool)        # Brzinsky-Fay
discrepancy_analysis()  # pseudo-R²

Visualization

Nine plot types, one grammar

Index plots, timelines, modal states, parallel coordinates — all return composable plotnine.ggplot objects. Compose with + exactly like ggplot2 in R.

p = index_plot(pool)
p + theme_bw()   # fully composable
